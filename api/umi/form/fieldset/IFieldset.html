Interface

umi\form\fieldset\IFieldset

interface IFieldset implements IFormEntity, Traversable

Интерфейс группы полей.

Группа полей, в понимании данного компонента, набор разнотипных элементов: IElement или IFieldset. Другими словами группа может хранить в себе элементы, группы полей либо под-формы.
