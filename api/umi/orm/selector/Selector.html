public ISelector types(array $typeNames)

Уточняет выбираемые типы данных.

Parameters

array $typeNames массив имен типов. Если после имени типа будет указан asterisk (), будут выбраны так же все дочерние типы. Пример: 'notebook' - будет выбран тип notebook и все дочерние от него, '*' - будут выбран все типы, включая базовый)
