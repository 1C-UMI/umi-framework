interface IModel

Интерфейс MVC-модели.

Модели, имплементирующие данный интерфейс автоматически внедряются в конструктор контроллера. Фактически, модель не обязательно должна имплементировать этот интерфейс. В случае, когда вы хотите создать модель не имплементирующую данный интерфейс, вам необходимо будет вызвать метод getModel() у контроллера.
