umi\dbal\builder\BaseQueryBuilder

abstract class BaseQueryBuilder implements IQueryBuilder, ILocalizable

Базовый построитель запросов.

Все построители запросов должны быть унаследованы от него, содержит шаблонные методы для всех построителей.
