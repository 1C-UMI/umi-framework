Class

umi\route\type\RegexpRoute

class RegexpRoute extends BaseRoute implements ILocalizable

Правило маршрутизатора на основе регулярных выражений.

Важно: части регулярного выражения, необходимые в запросе, требуется именовать(к примеру (?P\d+). Валидация неименованых частей работает, но данные из них не будут возвращены.
