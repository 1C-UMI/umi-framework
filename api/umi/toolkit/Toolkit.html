public IToolkit register(string $serviceInterfaceName, callable $builder)

Регистрирует сервис.

Каждый раз при обращении к сервису через IToolkit::get(), будет вызван билдер для получения экземпляра сервиса. Пример: $toolkit->register('umi\mail\IMail', function($concreteClassName, IToolkit $toolkit) { if ($concreteClassName) { return new $concreteClassName(); } else { return new MyDefaultMail(); } }

Parameters

string $serviceInterfaceName интерфейс сервиса
callable $builder билдер сервиса, создающий его экземпляр.

Return Value

IToolkit

Exceptions

AlreadyRegisteredException если сервис с указанным интерфейсом был зарегистрирован ранее

public IToolkit registerInjector(string $servicingInterfaceName, callable $injector)

Регистрирует инжектор для указанного интерфейса.

Инжектор может внедрять известные ему зависимости в объект. Пример: $toolkit->registerInjector('umi\logger\ILoggerAware', function($object, IToolkit $toolkit) { if ($object instanceof umi\logger\ILoggerAware) { $loggerService = $toolkit->get('Psr\Log\LoggerInterface'); $object->setLogger($loggerService); } }

Parameters

string $servicingInterfaceName имя обслуживаемого интерфейса
callable $injector

Return Value

IToolkit

Exceptions
