umi\authentication

Введение

Компонент umi\authentication предоставляет API для работы с аутентификацией.

Данный компонент оперирует несколькими сущностями, представленными на рисунке:

Данные аутентификации находятся в провайдере(Provider) После этого, эти данные попадают в адаптер(Adapter), который непосредственно и выполняет проверку введенных данных. Если данные верны, то результат работы адаптера содержит нетипизированный идентификатор - identity. Этот идентификатор сохраняется в хранилище(Storage). Всем процессом аутентификации занимается менеджер(Manager).

Tip

Аутентификацию не следует путать с авторизацией. Аутентификация - это процедура проверки подлинности пользователя. Авторизация - процесс наделения пользователя определенными правами.

Базовое использование

Для аутентификации вам необходимо создать менеджер аутентификации и установить в него провайдер аутентификации.

$manager = $authenticationFactory->createManager(); // создает менеджер аутентификации

// Создаем провайдер аутентификации
$provider = $authenticationFactory->createProvider(
    IAuthenticationFactory::PROVIDER_SIMPLE,
    [
        'username' => 'test',
        'password' => 'test'
    ]
);

$result = $manager->authenticate($provider); // аутентификация

if (!$result->isSuccessful()) {
    print('Wrong authentication.' . PHP_EOL);
} else {
    print('Successful authentication' . PHP_EOL);
}

var_dump($manager->isAuthenticated()); // возвращает true

$manager->forget(); // очищает данные аутентификации

var_dump($manager->isAuthenticated()); // возвращает false

Расширенное использование

Вы также можете настроить менеджер аутентификации при создании, изменив ему хранилище или адаптер:

// Создаем адаптер аутентификации
$adapter = $authenticationFactory->createAdapter(
    IAuthenticationFactory::ADAPTER_SIMPLE,
    [
        SimpleAdapter::OPTION_ALLOWED_LIST => [
            'user' => 'password'
        ]
    ]
);

// Создаем хранилище аутентификации
$storage = $authenticationFactory->createStorage(IAuthenticationFactory::STORAGE_SIMPLE);

$manager = $authenticationFactory->createManager([], $adapter, $storage); // создает менеджер аутентификации

Результат аутентификации (umi\authentication\result\IAuthResult) содержит в себе подробный результат аутентификации и идентификатор. При необходимости эти данные могут быть получены:

if ($result->isSuccessful()) {
    var_dump($result->getIdentity());
} else {
    throw new \Exception($result->getStatus());
}

Caution

Ни в коем случае не используйте проверку $result->getStatus() == IAuthResult::SUCCESSFUL для проверки успешности аутентификации. Это может повлечь за собой проблемы при добавлении новых статусов в систему.
