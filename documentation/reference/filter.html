umi\filter

Введение

Компонент umi\filter предназначен для фильтрации значений. Фильтрация - это преобразование значений. Фильтрация выполняется с помощью фильтров - правил преобразования значений.

Фильтр

umi\filter\IFilter - фильтр, основная сущность компонента umi\filter. Он выполняет преобразование одного значения в другое.

Коллекция фильтров

umi\filter\IFilterCollection - коллекция фильтров. Коллекция фильтров последовательно выполняет фильтрацию переданного ей значения с помощью содержащихся в ней фильтров. Она также имеет методы для изменения содержащихся в ней фильтров в процессе выполнения приложения.

Использование

Использование фильтров

Рассмотрим простейший пример использования фильтра:

$filter = $filterFactory
    ->createFilter(IFilterFactory::TYPE_STRING_TRIM);
var_dump($filter->filter('   string   '));

В этом коде фильтр используется для обрезки пробелов на конце переданной строки.

Большинство фильтров также имеют опции, которые могут быть переданы им при создании. К примеру:

$filter = $filterTools->createFilter(IFilterTools::TYPE_BOOLEAN, [
    'optional_values' => ['False' => false]
]); // создаем фильтр
var_dump($filter->filter('False')); // фильтруем значение, выведет bool(false)

В данном примере, фильтру был установлен список доп. значений для преобразования.

Использование коллекций фильтров

В реальных приложениях нам, чаще всего, недостаточно применения одного фильтра. Для этого в компоненте umi\filter существует коллекция фильтров. Рассмотрим пример использования коллекции фильтров:

$filter = $filterFactory->createFilterCollection([
        IFilterFactory::TYPE_STRING_TRIM => [],
        IFilterFactory::TYPE_BOOLEAN => [
            'optional_values' => ['yes' => true, 'no' => false]
        ]
    ]);

var_dump($filter->filter('   yes   ')); // true
var_dump($filter->filter('   no   ')); // false

В данном примере последовательно применяются фильтры обрезки пробелов строки и преобразования к boolean.

Важно

Фильтры в коллекции выполняются строго в указанном в конфигурации порядке.

Коллекция фильтров umi\filter\IFilterCollection позволяет дополнить список содержащихся в ней фильтров в любой момент времени. Для этого, у коллекции фильтров существуют методы изменения своего содержимого:

$filterCollection
    ->appendFilter($myFilter)
    ->prependFilter($mySecondFilter);

Типы фильтров

В компоненте umi\filter содержится необходимый набор фильтров.

umi\filter\toolbox\IFilterTools::TYPE_BOOLEAN

Фильтр преобразует значение в boolean(true или false) используя стандартное приведение типов php.

Список поддерживаемых опций:

optional_values => [ключ => значение, ...]

Список дополнительных значений, которые требуется привести к типу boolean. Валидатор будет приводить "ключ" в "значение".

umi\filter\toolbox\IFilterTools::TYPE_HTML_ENTITIES

Фильтр преобразует все возможные символы в соответствующие им HTML-сущности.

Список поддерживаемых опций:

flags

Битовая маска из нижеуказанных флагов, определяющих режим обработки кавычек, некорректных кодовых последовательностей и используемый тип документа. Cм. также htmlentities.

encoding

Кодировка, используемая по умолчанию. Cм. также htmlentities.

umi\filter\toolbox\IFilterTools::TYPE_INT

Фильтр преобразует значение к типу integer.

Список поддерживаемых опций:

base

Основание системы исчисления для преобразования. См. также intval

umi\filter\toolbox\IFilterTools::TYPE_NULL

Фильтр преобразует значение к NULL.

Список поддерживаемых опций:

optional_values => [значение, ...]

Дополнительные значения для преобразования. Значения, указанные в этой опции будут преобразованны в NULL.

umi\filter\toolbox\IFilterTools::TYPE_REGEXP

Фильтр преобразует значение используя указанное регулярное выражение.

Список поддерживаемых опций:

pattern

Обязательный параметр. Регулярное выражение, по которому будет осуществляться преобразование. См. также preg_replace.

replacement

Обязательный параметр. Значение, используемое для замены. См. также preg_replace.

limit

Ограничение количества производимых замен. См. также preg_replace.

umi\filter\toolbox\IFilterTools::TYPE_STRING_TO_LOWER

Фильтр преобразует все символы строки к нижнему регистру.

Список поддерживаемых опций:

encoding

Кодировка.

umi\filter\toolbox\IFilterTools::TYPE_STRING_TO_UPPER

Фильтр преобразует все символы строки к верхнему регистру.

Список поддерживаемых опций:

encoding

Кодировка.

umi\filter\toolbox\IFilterTools::TYPE_STRING_TRIM

Фильтр удаляет пробельные или другие символы из начала и конца строки.

Список поддерживаемых опций:

charlist => строка

Cписок символов для удаления.

umi\filter\toolbox\IFilterTools::TYPE_STRIP_NEW_LINES

Фильтр преобразует символы переноса строки в пробелы.

umi\filter\toolbox\IFilterTools::TYPE_STRIP_TAGS

Фильтр вырезает HTML тэги из строки.

Список поддерживаемых опций:

allowed => [значение, ...]

Тэги, которые требуется оставить в результирующей строке.
