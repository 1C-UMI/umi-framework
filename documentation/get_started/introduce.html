Введение

Этот раздел документации предназначен для того, чтобы дать краткий обзор возможностей UMI.Framework на примере создания простого приложения с использованием базы данных и иерархической MVC архитектуры.

Подразумевается, что для вашего приложения вы используете:

  • PHP версии не ниже 5.4 (рекомендуется использовать 5.5)

  • MySQL версии не ниже 5.0, доступный через расширение PDO

  • Apache

Также вы можете использовать другие конфигурации аналогичные данной.

Структура демонстрационного приложения

Приложение, которое мы будем создавать - это простой блог, который позволяет просматривать, создавать, редактировать и удалять посты.

Для реализации данного функционала нам понадобятся следующие страницы:

Table 1. Возможности демонстрационного приложения

СтраницаОписание
Главная страницаСтатическая страница с информацией о приложении
Список постов блогаВыводит список постов блога, выводит ссылки для добавления нового поста и удаления существующих
Добавление поста в блогВыводит и обрабатывает форму для добавления нового поста в блог
Редактирование поста блогаВыводит и обрабатывает форму для редактирования поста в блоге
Удаление поста блогаВыводит и обрабатывает форму для подтверждения удаления поста
