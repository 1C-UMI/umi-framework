Interface

umi\config\entity\ISeparateConfigSource

interface ISeparateConfigSource implements IConfigSource

Отделенная конфигурация, загружаемая при необходимости.

С помощью данного типа конфигурации реализуется Lazy конфигурационные файлы, а также подключаемые удаленные конфигурации и т.п.
