public IEventObservant bindEvent(string $eventType, callable $eventHandler, array $tags = null)

Устанавливает обработчик события

Parameters

string $eventType тип события (уникальный строковый идентификатор)
callable $eventHandler callable обработчик события
array $tags тэги, с которыми должно происходить событие, на которое устанавливается обработчик. Тэги позволяют подписаться на события, которые происходят с конкретными объектами. Если тэги не указаны, обработчик будет вызван для всех объектов, поднимающих события.

Return Value
